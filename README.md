# Personal Portfolio

A modern, responsive personal portfolio built with React, Vite, and Tailwind CSS. The portfolio features a sleek dark theme, smooth animations, and a clean layout to showcase projects, skills, and contact information.

## Features

- **Modern Tech Stack:** React 19, Vite, and Tailwind CSS v4.
- **Smooth Animations:** Integrated with [Framer Motion](https://www.framer.com/motion/) for fluid page transitions and scroll animations.
- **Crisp Iconography:** Utilizing [Lucide React](https://lucide.dev/) for consistent, lightweight icons.
- **Interactive UI:** Includes `canvas-confetti` for fun, interactive elements.
- **Responsive Design:** Fully responsive layout that looks great on desktop, tablet, and mobile devices.
- **Dark Theme Environment:** Sleek `#0a0a12` based dark mode by default for an elegant viewing experience.

## Project Structure

- `Navbar`: Responsive navigation across the site.
- `Hero`: Eye-catching introductory section.
- `About`: Detailed section covering background and skills.
- `Works`: Showcase of past projects and accomplishments.
- `Contact`: Information and forms for getting in touch.
- `Footer`: Links and secondary navigation.

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jesselsajulga/React-TailwindCSS.git
   cd React-TailwindCSS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview the production build:**
   ```bash
   npm run preview
   ```
